HTML basics

Tim Berners-Lee

In this course you will learn the basic principles of the foundational web language HTML. HTML is the backbone of all websites and is where the content you see is defined. Written in 1990 by Tim Berners-Lee HTML was comprised of 18 element tags which most are still utilized in the most recent versions of HTML. If you are just starting your journey to become a website developer HTML is the perfect place to start learning. You will learn how to make a simple website and even add a little style to your elements in this course. Click Next Lesson to start.
